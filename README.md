# 🔐 Password Strength Checker

A Flask-based web application that evaluates password strength and provides security recommendations.

## Features
- Password strength analysis (Weak / Medium / Strong)
- Suggests improvements
- Prevents common passwords
- SHA-256 hashing (cybersecurity concept)
- Random password generator

## Tech Stack
- Python
- Flask
- HTML/CSS

## Run Project
```bash
pip install flask
python app.py
